# 📊 Excel Practical – Advanced Functions & Data Analysis

## 🚀 Project Overview

This project demonstrates advanced **Microsoft Excel techniques** for data analysis, automation, and visualization.
It covers statistical functions, dynamic formulas, formatting, and data summarization methods used in real-world analytics.

---

## 🎯 Objectives

* Apply advanced Excel formulas for data analysis
* Automate calculations using dynamic functions
* Perform statistical analysis on datasets
* Improve data visualization and presentation

---

## 🧠 Topics Covered

---

## 📌 1. Custom Formatting

### 🔹 Description

Custom Formatting is used to change the appearance of data without modifying actual values.

### 🔹 Examples

* Display currency symbols (₹, $, £)
* Add arrows (▲ ▼) for trends
* Highlight values using colors

---

## 📌 2. Timestamp in Excel

### 🔹 Description

Timestamp records the **date and time** when an event occurs.

### 🔹 Methods

* Using `NOW()` and `TODAY()`
* Using IF formula with iterative calculation
* Manual shortcuts:

  * Date → `Ctrl + ;`
  * Time → `Ctrl + Shift + ;`

---

## 📌 3. GROUPBY Alternatives

### 🔹 Description

Since GROUPBY is limited to Excel 365, alternatives are used:

### 🔹 Methods

#### ✅ Pivot Table

* Quick data summarization
* Drag-and-drop interface

#### ✅ SUMIF

```
=SUMIF(range, condition, sum_range)
```

#### ✅ UNIQUE + SUMIF

* Extract unique values
* Calculate totals dynamically

---

## 📌 4. Logical & Dynamic Functions

### 🔹 Functions Covered

* `LARGE()` → Top values
* `SMALL()` → Lowest values
* `ROWS()` → Count records
* `SEQUENCE()` → Generate series
* `INDEX()` → Retrieve values

---

## 📌 5. Statistical Functions

### 🔹 Mean, Median, Mode

* `AVERAGE()` → Mean
* `MEDIAN()` → Middle value
* `MODE.SNGL()` → Most frequent value

---

### 🔹 Standard Deviation

* `STDEV.P()` → Entire population
* `STDEV.S()` → Sample data

👉 Measures variation in data

---

### 🔹 Quartile & Percentile

* `QUARTILE()` → Divide data into 4 parts
* `PERCENTILE()` → Divide data into 100 parts

👉 Helps in distribution analysis

---
